CS 240: Computer Architecture and Assembly Language

An introduction to computer architecture, including binary representation, digital logic, processor organization, memory hierarchy, and instruction-level parallelism. Students will write programs in assembly language to develop a practical understanding of how high-level code is translated into machine instructions. Topics include cache design, pipelining, and an introduction to operating system concepts such as virtual memory and process scheduling. Prerequisite: CS 230 with a grade of C or better.
